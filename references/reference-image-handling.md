# Reference Image Handling

Use this file when the user provides or mentions a reference image.

The goal is to avoid vague instructions like “make it like this image.” Instead, identify exactly which layer of the reference image should be used.

## First reference question

Ask: “What do you mainly want to borrow from the reference image?”

Options:

- Overall feeling
- Composition and layout
- Color palette
- Lighting
- Subject pose or expression
- Material or texture
- Typography or title placement
- Visual density and spacing
- Several of these — I will explain
- I am not sure — please analyze it for me

## If the user says “overall feeling”

Ask: “When you say the feeling of the reference image, which part matters most?”

Options:

- The color feels right
- The lighting feels right
- The image feels premium
- The composition feels balanced
- The subject looks natural
- The mood feels quiet or emotional
- The layout feels clean
- I am not sure — please decide
- Other — I will explain

## If the user says “composition”

Ask: “What part of the composition should we keep?”

Options:

- Where the main subject is placed
- How much empty space there is
- The relationship between foreground and background
- The way text and subject are arranged
- The sense of depth and layers
- The overall visual balance
- I am not sure — please analyze it
- Other — I will explain

## If the user says “person expression or pose”

Ask: “What part of the person’s state should we preserve?”

Options:

- Facial expression
- Body pose
- Emotional state
- Overall personality
- Styling and outfit feeling
- Relationship with the environment
- I want several of these
- I am not sure — please analyze it

## If the user says “premium feeling”

Ask: “When you say premium, which kind of premium do you mean?”

Options:

- Simple and restrained
- Soft and elegant
- Dark and cinematic
- Luxury and polished
- Clean and Apple-like
- Editorial and fashion-like
- High-tech but not cheap
- I am not sure — please choose
- Other — I will describe it

## Reference image rules

- Do not blindly copy the reference image.
- Do not copy specific faces, logos, copyrighted characters, or private details unless allowed and appropriate.
- Separate what should be borrowed from what should be changed.
- If the user says “same feeling”, break it down into color, lighting, composition, mood, material, and layout.
- If the user says “same composition”, break it down into subject placement, empty space, depth, foreground/background, and text placement.
- If the user says “same style”, ask whether style means color, texture, medium, lighting, layout, or mood.
